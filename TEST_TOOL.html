🔍 插件检测工具

5秒验证插件是否正常工作

📋 使用步骤

  1. 在下面复制检测代码
  2. 打开AI网站（DeepSeek、豆包或千问）
  3. 按F12打开开发者工具
  4. 切换到Console标签
  5. 粘贴代码并回车
  6. 查看检测结果

1️⃣ 完整检测脚本（推荐）

console.log('=== 插件检测 ==='); 
const hostname = window.location.hostname; console.log('平台:', hostname); const input = document.querySelector('textarea') || document.querySelector('div[contenteditable="true"]'); console.log('输入框:', input ? '✅ 找到' : '❌ 未找到'); const buttons = document.querySelectorAll('button'); console.log('按钮数量:', buttons.length, buttons.length > 0 ? '✅ 找到' : '❌ 未找到'); const svg = document.querySelector('svg'); console.log('SVG图标:', svg ? '✅ 找到' : '❌ 未找到'); console.log('=== 检测完成 ===');

2️⃣ 快速检测

// 检测输入框

3️⃣ 按钮检测

// 检测所有按钮

// 检测插件是否注入

✅ 成功标准

如果在AI网站运行检测脚本后看到：

  • 平台: 显示域名
  • 输入框: ✅ 找到
  • 按钮数量: ✅ 找到（大于0）
