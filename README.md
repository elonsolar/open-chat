# 多模型AI对话助手 - Edge浏览器插件

通过**操作AI网页版**实现免费对话，无需API密钥！

## ⭐ 核心特性

- 🎯 **完全免费** - 无需API密钥，通过网页版实现
- 🤖 **多AI支持** - DeepSeek、豆包、ChatGPT、千问
- 👥 **角色管理** - 创建自定义角色，配置提示词
- 💬 **会话管理** - 多AI同时回答，对比结果
- 🔗 **会话绑定** - 自动保存会话URL，保持上下文
- 📱 **侧边栏** - 便捷操作，不影响浏览
- 💾 **本地存储** - 数据不上传，隐私安全

## 📦 安装

1. 打开 `edge://extensions/`
2. 启用"开发人员模式"
3. 点击"加载解压缩的扩展"
4. 选择项目根目录

## 🚀 快速开始

### 1. 登录AI网站
- DeepSeek: https://chat.deepseek.com
- 豆包: https://www.doubao.com/chat
- ChatGPT: https://chatgpt.com
- 千问: https://www.qianwen.com

### 2. 创建角色
1. 打开插件侧边栏 → 切换到"角色"
2. 点击"新建角色"
3. 填写信息：
   - 角色名称：如"Python助手"
   - 服务提供商：选择AI平台
   - 系统提示词：如"你是Python专家"
4. 点击"创建"并"测试"

### 3. 开始对话
1. 切换到"会话" → 点击"新建会话"
2. 填写信息：
   - 会话名称：如"学习Python"
   - 选择角色：可多选
   - 上下文模式：AI自保持（推荐）
3. 点击"创建" → 输入消息 → 多AI同时回答！

## 💡 核心功能

### 会话URL绑定
- 每个角色自动保存专属会话
- 保持对话历史和上下文
- 即使标签页关闭也能恢复

**示例：**
```
第1次：问Python安装 → 自动保存会话
第2次：问Python语法 → 继续之前的会话（记得第1次的内容）
第3次：问代码优化 → 记得所有对话
```

### 多AI同时对话
- 一个问题，多个AI同时回答
- 对比不同AI的答案
- 选择最适合的回答

## 🐛 常见问题

**Q: 测试失败？**
A: 确保已登录AI网站，刷新页面后重试

**Q: AI不回复？**
A: 检查网络、刷新AI页面、重新测试连接

**Q: 数据会丢失？**
A: 存储在本地，清除浏览器数据会丢失

**Q: 为什么不用API？**
A: API需要付费，网页版免费
