# 多模型AI对话助手

免费多模型对话，无需 API 密钥。通过操作 AI 网页版实现多模型协作。

## 功能特性

- 🚀 **多平台支持** - DeepSeek、千问、豆包、ChatGPT
- 🤖 **角色管理** - 自定义系统提示词，为每个会话设置昵称和额外提示
- 🔄 **发送模式** - 并行/顺序接龙/随机接龙
- 💾 **上下文模式** - 自有上下文/共享上下文（不可变）
- 🎯 **自动检测** - 智能检测 AI 回复完成，支持代码块渲染

## 安装

1. 下载项目到本地
2. 打开 `edge://extensions/` 或 `chrome://extensions/`
3. 启用"开发者模式"
4. 点击"加载已解压的扩展程序"，选择项目根目录

## 快速开始

### 1. 登录 AI 平台

在浏览器中登录对应平台（需保持页面打开）：

| 平台 | 网址 |
|------|------|
| DeepSeek | https://chat.deepseek.com |
| 千问 | https://www.qianwen.com |
| 豆包 | https://www.doubao.com/chat |
| ChatGPT | https://chatgpt.com |

### 2. 创建角色

侧边栏 → 角色 → 新建角色
- 填写名称、选择提供商
- 设置系统提示词
- 可选：设置昵称和额外提示（针对具体对话优化）

### 3. 创建会话

侧边栏 → 会话 → 新建会话
- 选择多个角色
- 选择上下文模式（自有/共享，创建后不可更改）
- 开始对话

## 发送模式

- **并行** - 所有角色同时收到消息，独立回答
- **顺序接龙** - 按顺序依次发送，每个角色看到前面的回答
- **随机接龙** - 随机打乱顺序后接龙

## 常见问题

**Q: 测试失败？**  
A: 确保已登录 AI 平台，刷新页面重试

**Q: AI 不回复？**  
A: 检查网络连接，确保 AI 页面已打开

**Q: 数据会丢失吗？**  
A: 数据存储在浏览器本地，清除浏览器数据会丢失

**Q: 代码块不显示？**  
A: 使用 `/mode` 切换发送模式或刷新页面
