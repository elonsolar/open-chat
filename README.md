# 多模型AI对话助手

通过操作 AI 网页版实现免费多模型对话，无需 API 密钥。

## 安装

1. 打开 `edge://extensions/`
2. 启用开发人员模式
3. 加载解压缩的扩展 → 选择项目根目录

## 快速开始

### 1. 登录 AI 网站

| 平台 | 网址 |
|------|------|
| DeepSeek | https://chat.deepseek.com |
| 豆包 | https://www.doubao.com/chat |
| ChatGPT | https://chatgpt.com |
| 千问 | https://www.qianwen.com |

### 2. 创建角色

侧边栏 → 角色 → 新建角色 → 填写名称、提供商、系统提示词 → 创建

### 3. 创建会话

侧边栏 → 会话 → 新建会话 → 选角色、选上下文模式 → 创建 → 发消息

## 平台支持

所有平台均需保持对应网页在浏览器中打开。

## 命令

在会话输入框发送：

| 命令 | 说明 |
|------|------|
| `/mode` | 选择发送模式：并行 / 顺序接龙 / 随机接龙，并拖拽调整角色顺序 |
| `/clear` | 清除会话消息 |

发送模式说明：
- **并行** — 所有角色同时收到问题，各自独立回答
- **顺序接龙** — 按角色顺序依次发送，每个角色能看到前面角色的回答
- **随机接龙** — 随机打乱顺序后依次接龙

## FAQ

**测试失败？** 确认已登录 AI 网站，刷新页面重试。

**AI 不回复？** 检查网络，刷新 AI 页面，重新测试连接。

**数据会丢失？** 数据存储在浏览器本地，清除浏览器数据会丢失。
