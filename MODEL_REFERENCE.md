## 💡 使用技巧

### 技巧1：模型字段可以留空
```
选择提供商后，模型会自动填充默认值
直接点击"创建"即可使用
```

### 技巧2：手动指定模型
```
如果需要使用特定模型，可以手动修改：
例如：deepseek-coder（用于编程）
```

### 技巧3：创建不同专长的角色
```
角色1：DeepSeek编程助手
  - 模型：deepseek-coder
  - 提示词：专注于编程问题

角色2：DeepSeek对话助手
  - 模型：deepseek-chat
  - 提示词：日常对话
```

## ⚠️ 注意事项

### 1. 模型名称大小写
```
✅ 正确：deepseek-chat
❌ 错误：DeepSeek-Chat
```

### 2. 模型名称格式
```
✅ 正确：deepseek-chat, qwen-plus
❌ 错误：deepseek chat, qwen plus
```

### 3. 默认模型
```
如果不确定使用哪个模型，直接使用自动填充的默认值即可
```

## 🔍 常见问题

### Q1: 我不知道该填什么模型？

**A**: 直接留空即可！系统会自动填充推荐的默认值。

### Q2: 模型名称填错了怎么办？

**A**: 目前插件只做记录，不影响实际使用。真正使用的是网页版默认模型。

### Q3: 不同平台模型可以混用吗？

**A**: 可以！你可以创建多个不同平台的角色，在一个会话中同时使用。

### Q4: 如何知道哪个模型最好？

**A**: 参考"快速选择指南"，或直接使用默认值。
