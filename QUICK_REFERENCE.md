# 🎯 快速参考卡

## ⚡ 30秒快速开始

```
1️⃣ 登录 AI 网站
   → https://chat.deepseek.com

2️⃣ 创建角色
   → 侧边栏 → 角色 → 新建 → 选择DeepSeek

3️⃣ 测试连接
   → 点击角色卡片上的"测试"按钮

4️⃣ 创建会话
   → 侧边栏 → 会话 → 新建 → 勾选角色

5️⃣ 开始对话
   → 输入消息 → 发送 → 等待回复
```

---

## 📍 各个功能在哪里？

| 功能 | 位置 | 说明 |
|------|------|------|
| **打开侧边栏** | 点击插件图标 → "打开侧边栏" | 主操作界面 |
| **创建角色** | 侧边栏 → "角色"标签 → "新建角色" | 选择AI平台 |
| **测试连接** | 角色卡片 → "测试"按钮 | 验证是否可用 |
| **创建会话** | 侧边栏 → "会话"标签 → "新建会话" | 开始对话 |
| **选择角色** | 创建会话时勾选角色 | 选择使用哪些AI |
| **发送消息** | 聊天页面输入框 → "发送"按钮 | 与AI对话 |

## ⚠️ 必须做好的3件事

### 1️⃣ 登录AI网站

**位置**：新标签页

**网址**：
- DeepSeek: https://chat.deepseek.com
- 豆包: https://www.coze.com
- 千问: https://tongyi.aliyun.com

**确认**：能看到聊天界面

---

### 2️⃣ 创建角色

**位置**：侧边栏 → 角色标签

**步骤**：
1. 点击"新建角色"
2. 填写信息
3. **重要**：在"服务提供商"选择AI平台
4. 点击"创建"

**示例**：
```
名称：DeepSeek助手
提供商：DeepSeek  ← 关键！选择AI平台
模型：deepseek-chat
提示词：你是一个友好的助手
```

---

### 3️⃣ 测试连接

**位置**：角色卡片 → "测试"按钮

**目的**：确保插件能连接到AI网站

**成功标志**：弹出"连接成功"提示

**失败处理**：
1. 检查是否登录AI网站
2. 刷新AI网站页面
3. 重新测试

## 💬 对话流程

```
创建会话时选择角色
        ↓
    勾选至少一个角色
        ↓
      点击创建
        ↓
    聊天页面打开
        ↓
    输入消息
        ↓
    点击发送
        ↓
    等待10-30秒
        ↓
    AI回复出现
```

## 🚨 常见错误速查

| 错误 | 原因 | 解决方案 |
|------|------|---------|
| "未选择角色" | 创建会话时没勾选角色 | 重新创建，勾选角色 |
| "发送中..."一直显示 | AI网站未打开或未登录 | 打开并登录AI网站 |
| "测试失败" | 未登录AI网站 | 登录后重新测试 |
| "侧边栏没反应" | 插件未加载 | 刷新插件或重启浏览器 |
| "找不到元素" | AI网站页面变化 | 更新选择器（高级） |
