# 图标目录使用说明

## 快速生成图标（推荐）

### Windows用户
双击运行 `生成图标.bat` 文件，将自动在浏览器中打开图标生成器。

### 手动打开
在浏览器中打开 `generator.html` 文件。

## 图标生成器使用步骤

1. **打开生成器**
   - 双击 `生成图标.bat`（Windows）
   - 或直接在浏览器中打开 `generator.html`

2. **自定义图标（可选）**
   - 调整图标主色调
   - 修改图标文字（建议1-2个字符）
   - 调整背景颜色

3. **下载图标**
   - 点击"下载所有图标"按钮
   - 会自动下载3个PNG文件

4. **放置图标**
   - 将下载的文件移动到当前目录（`icons/`）
   - 文件名应该是：
     - icon16.png
     - icon48.png
     - icon128.png

5. **刷新插件**
   - 在Edge浏览器中打开 `edge://extensions/`
   - 找到"多模型AI对话助手"插件
   - 点击刷新按钮🔄

## 验证图标

图标添加成功后，插件应该显示：
- 浏览器工具栏中的图标
- 插件列表中的图标
- 插件详情页的大图标

## 其他选项

如果不想使用生成器，可以：
1. 使用在线工具生成图标
2. 手动创建PNG图像
3. 使用默认图标（插件仍然可以工作）

## 文件说明

- `generator.html` - 图标生成器页面
- `生成图标.bat` - Windows快捷启动脚本
- `icon.svg` - SVG格式的源图标
- `README.txt` - 本说明文件
- `icon16.png` - 16x16像素图标（需要生成）
- `icon48.png` - 48x48像素图标（需要生成）
- `icon128.png` - 128x128像素图标（需要生成）
