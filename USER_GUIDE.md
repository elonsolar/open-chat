# 📖 新手使用指南 - 5分钟上手

## 🎯 你需要完成的任务

1. ✅ 登录AI网站
2. ✅ 创建角色（选择AI平台）
3. ✅ 测试连接
4. ✅ 创建会话（选择角色）
5. ✅ 开始对话

---

## 🚀 详细步骤（跟着做就行）

### 第1步：登录AI网站（必须先做！）

**为什么**：插件通过操作AI网页版来实现对话，所以必须先登录。

**操作**：
```
1. 打开新标签页
2. 访问并登录你想使用的AI平台：

   DeepSeek:  https://chat.deepseek.com
   豆包:      https://www.doubao.com/chat/
   千问:      https://www.qianwen.com
   ChatGPT:   https://chatgpt.com

3. 确保已经登录（可以看到聊天界面）
4. 保持这个标签页打开（不要关闭）
```

**重要**：
- ⚠️ 必须先登录，否则插件无法操作
- ⚠️ 登录后不要关闭标签页
- ⚠️ 可以登录多个平台

---

### 第2步：打开插件侧边栏

**操作**：
```
1. 在Edge浏览器工具栏找到插件图标
2. 点击图标
3. 在弹出窗口中点击"打开侧边栏"按钮
```

**如果没反应**：
```
1. 按 Ctrl+Shift+S 打开侧边栏
2. 或者在地址栏输入：edge://sidebar
```

**你应该看到**：
```
┌─────────────────────┐
│  AI对话助手         │
│  ┌─────┬─────┬─────┐│
│  │会话 │角色 │设置 ││
│  └─────┴─────┴─────┘│
└─────────────────────┘
```

---

### 第3步：创建角色（选择AI平台）

**什么是角色**？
- 角色就是一个AI助手
- 每个角色对应一个AI平台（DeepSeek、豆包等）
- 你可以创建多个角色，使用不同的AI

**操作**：
```
1. 在侧边栏中点击"角色"标签
2. 点击"新建角色"按钮
3. 填写表单：

   角色名称：DeepSeek助手
   服务提供商：DeepSeek ← 这里选择AI平台！
   模型：deepseek-chat
   系统提示词：你是一个友好的助手

4. 点击"创建"
```

**创建更多角色**（可选）：
```
重复上面的步骤，创建其他平台的角色：

- 角色名：豆包编程专家，提供商：豆包
- 角色名：千问老师，提供商：千问
- 角色名：ChatGPT助手，提供商：ChatGPT
```

---

### 第4步：测试连接（重要！）

**为什么测试**？
- 确保插件能成功连接到AI网站
- 避免发送消息时出错

**操作**：
```
1. 在角色列表中找到你刚创建的角色
2. 点击角色卡片上的"测试"按钮
3. 等待结果（几秒钟）
```

**结果**：
```
✅ 成功：弹出"连接成功"提示
   → 可以继续使用

❌ 失败：弹出"连接失败"提示
   → 检查：
      1. 是否已登录AI网站？
      2. AI网站标签页是否打开？
      3. 网络是否正常？
```

**如果失败**：
```
1. 打开对应的AI网站（如DeepSeek）
2. 确认已登录
3. 刷新页面
4. 重新点击"测试"按钮
```

---

### 第5步：创建会话（选择角色）

**什么是会话**？
- 会话就是一次对话
- 一个会话可以选择多个角色（多个AI）
- 发送消息时，所有选中的角色都会回答

**操作**：
```
1. 在侧边栏中点击"会话"标签
2. 点击"新建会话"按钮
3. 填写表单：

   会话名称：我的第一次对话
   选择角色：☑ DeepSeek助手 ← 勾选你创建的角色！

4. 点击"创建"
```

**多AI对话**（推荐试试）：
```
如果你创建了多个角色，可以都勾选：

会话名称：AI大比拼
选择角色：
  ☑ DeepSeek助手
  ☑ 豆包编程专家
  ☑ 千问老师

→ 发送消息时，3个AI会同时回答！
```

---

### 第6步：开始对话

**操作**：
```
1. 创建会话后，聊天页面会自动打开
2. 在输入框中输入消息，例如：
   "你好，请介绍一下自己"
3. 点击"发送"按钮
4. 等待AI回复（可能需要10-30秒）
```

**你应该看到**：
```
┌──────────────────────────┐
│ 我: 你好，请介绍一下自己  │
├──────────────────────────┤
│ DeepSeek助手:            │
│ 你好！我是DeepSeek...     │
│ [DeepSeek]               │
└──────────────────────────┘
```

**发送消息后一直显示"发送中..."？**
```
1. 检查AI网站标签页是否打开
2. 等待更长时间（AI思考需要时间）
3. 查看AI网站是否有错误提示
```

## ❓ 常见问题

### Q1: 侧边栏打不开？

**方案1**：使用快捷键
```
按 Ctrl+Shift+S
```

**方案2**：手动打开
```
1. 点击插件图标
2. 点击"打开侧边栏"按钮
```

**方案3**：重新加载插件
```
1. 打开 edge://extensions/
2. 找到插件
3. 点击刷新按钮
```

---

### Q2: 提示"未选择角色"？

**原因**：创建会话时没有勾选角色

**解决**：
```
1. 创建会话时
2. 在"选择角色"区域
3. 勾选至少一个角色
4. 然后点击"创建"
```

---

### Q3: 一直显示"发送中..."？

**检查清单**：
```
□ AI网站标签页是否打开？
□ 是否已登录AI网站？
□ 网络是否正常？
□ AI网站是否有错误？
```

**解决方法**：
```
1. 打开AI网站标签页
2. 确认页面正常
3. 等待更长时间（最多60秒）
```

---

### Q4: 测试连接失败？

**最常见原因**：未登录AI网站

**解决步骤**：
```
1. 打开 https://chat.deepseek.com
2. 登录账号
3. 确认能看到聊天界面
4. 回到插件
5. 重新点击"测试"按钮
```

---

### Q5: 不知道在哪设置角色？

**位置**：插件侧边栏 → 角色标签

**详细步骤**：
```
1. 打开插件侧边栏
2. 点击顶部"角色"标签（三个标签之一）
3. 点击"新建角色"按钮
4. 在"服务提供商"下拉框中选择AI平台
5. 填写其他信息
6. 点击"创建"
```
