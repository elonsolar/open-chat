## 🔧 使用方法

### 查看日志步骤

1. **打开AI网站**
   ```
   DeepSeek: https://chat.deepseek.com
   豆包: https://www.doubao.com/chat/
   千问: https://www.qianwen.com
   ```

2. **打开开发者工具**
   ```
   按F12
   或右键 → 检查
   ```

3. **切换到Console标签**
   ```
   找到"Console"标签
   点击进入
   ```

4. **创建角色并测试**
   ```
   侧边栏 → 角色标签 → 新建角色
   选择平台 → 创建
   点击"测试"按钮
   ```

5. **查看日志输出**
   ```
   在Console中查看带 [doubao] 或 [deepseek] 前缀的日志
   ```

## 🎯 诊断问题

### 问题1: 找不到输入框

**日志**：
```
[doubao] 元素查找超时: textarea
```

**解决**：
- 刷新页面
- 检查是否已登录
- 查看页面是否完全加载

---

### 问题2: Enter无效

**日志**：
```
[doubao] Enter事件结果: true
```

## 💡 根据日志排查

### 如果看到"未找到输入框"

**说明**：页面结构不匹配

**操作**：
1. 按F12
2. 切换到Elements标签
3. 点击元素选择器（箭头图标）
4. 点击输入框
5. 查看HTML结构
6. 告诉我实际的标签名和class

### 如果看到"找到输入框"但值为空

**说明**：输入失败

**操作**：
1. 刷新页面
2. 重新测试
3. 检查是否有权限限制

### 如果看到"消息应该已发送"但超时

**说明**：AI没有回复

**可能原因**：
- AI服务慢
- 网络问题
- 需要更长等待时间
- 页面有加载指示器未消失

**操作**：
- 查看页面是否有"正在思考"或"加载中"提示
- 手动在页面输入测试消息，看AI是否回复
- 检查网络连接
