# 🚀 快速开始指南

## 5分钟上手多模型AI对话助手

### 📦 第一步：安装插件

1. **生成图标（可选）**
   - 双击 `icons/生成图标.bat`
   - 点击"下载所有图标"
   - 将下载的3个PNG文件移动到 `icons/` 目录

2. **加载插件到Edge浏览器**
   - 打开Edge浏览器
   - 地址栏输入：`edge://extensions/`
   - 打开右上角"开发人员模式"
   - 点击"加载解压缩的扩展"
   - 选择项目文件夹 `open-chat`

### 🔑 第二步：登录AI网站

在新标签页中打开并登录你想使用的AI平台：

| 平台 | 网址 | 说明 |
|------|------|------|
| DeepSeek | https://chat.deepseek.com | 免费使用 |
| 豆包 | https://www.coze.com | 需要账号 |
| 千问 | https://tongyi.aliyun.com | 需要阿里账号 |
| ChatGPT | https://chatgpt.com | 需要订阅 |

**重要**：必须先登录，否则插件无法操作！

### 👤 第三步：创建角色

1. **打开插件**
   - 点击Edge工具栏的插件图标
   - 点击"打开侧边栏"

2. **切换到"角色"标签**

3. **点击"新建角色"**

4. **填写角色信息**
   - **角色名称**：`DeepSeek编程助手`
   - **服务提供商**：`DeepSeek`
   - **模型**：`deepseek-chat`
   - **系统提示词**：
     ```
     你是一个专业的编程助手，擅长解释代码、调试问题、提供编程建议。
     回答要简洁明了，必要时给出代码示例。
     ```

5. **点击"创建"**

6. **测试连接**（重要！）
   - 点击角色卡片上的"测试"按钮
   - ✅ 看到"连接成功"即可继续
   - ❌ 如果失败，检查是否已登录DeepSeek网站

### 💬 第四步：创建会话

1. **切换到"会话"标签**

2. **点击"新建会话"**

3. **填写会话信息**
   - **会话名称**：`编程学习`
   - **选择角色**：勾选刚才创建的角色

4. **点击"创建"**

5. **聊天页面会自动打开**

### 🎉 第五步：开始对话

在聊天页面的输入框中输入消息，例如：

```
如何用Python实现快速排序？
```

点击"发送"或按Enter键，等待AI回复！

## 💡 使用技巧

### 多AI同时回答

创建会话时选择多个角色，可以同时获得不同AI的回答：

1. 创建多个不同平台的角色（DeepSeek、豆包、千问）
2. 新建会话时勾选所有角色
3. 发送消息，所有AI都会同时回答

### 对比不同AI的回答

每个角色的回答会独立显示，方便对比：
- 查看不同AI的思路
- 选择最合适的答案
- 学习不同的表达方式

### 创建专业角色

为不同场景创建专门的角色：

**角色1：代码审查员**
```
提供商：DeepSeek
提示词：你是一个严格的代码审查员，专注于发现代码中的bug、
安全问题和性能问题。请指出问题并给出改进建议。
```

**角色2：英语教师**
```
提供商：ChatGPT
提示词：你是一位专业的英语教师，擅长语法纠错和写作指导。
请纠正我的英语错误，并解释原因。
```

**角色3：创意写作助手**
```
提供商：千问
提示词：你是一位创意写作专家，擅长构思故事情节、角色设定
和场景描写。请帮助我进行创意写作。
```

## ⚠️ 常见问题

### Q1: 提示"测试失败"怎么办？

**解决方案：**
1. 确保已在浏览器中登录对应AI网站
2. 检查网络连接
3. 刷新AI网站页面
4. 重新点击"测试"按钮

### Q2: 发送消息后没有回复？

**可能原因：**
- AI网站标签页被关闭了
- 网络延迟
- AI服务暂时不可用

**解决方案：**
1. 检查AI网站标签页是否打开
2. 等待更长时间（最多60秒）
3. 刷新页面重试

### Q3: 想添加更多AI平台？

需要修改代码，参考：
- `utils/platform-adapter.js` - 添加平台选择器
- `manifest.json` - 添加content_scripts规则
