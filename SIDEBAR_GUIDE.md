# 🔧 侧边栏问题完整解决方案

## 问题：打开侧边栏失败

### 错误信息
```
TypeError: Error in invocation of sidePanel.open():
No matching signature.
```

### 原因
Chrome Extension API 的 `chrome.sidePanel.open()` 需要指定窗口ID参数。

---

## ✅ 解决方案

### 方案1：使用快捷键（推荐）

**快捷键**：`Ctrl + Shift + S`

这是最简单快捷的方式！

---

### 方案2：通过浏览器工具栏

**步骤**：
```
1. 看浏览器右上角工具栏
2. 找到扩展图标（拼图图标）
3. 点击后找到"多模型AI对话助手"
4. 选择"打开侧边栏"或类似选项
```

---

### 方案3：通过地址栏

**在地址栏输入**：
```
edge://sidebar
```

然后选择你的插件。

---

### 方案4：右键点击工具栏

**步骤**：
```
1. 在浏览器工具栏空白处右键
2. 选择"显示侧边栏"
3. 选择扩展程序
4. 找到"多模型AI对话助手"
```

## 📋 侧边栏三个标签的用途

### 1. 会话标签

**功能**：
- 查看所有对话
- 创建新会话
- 删除会话

**操作**：
```
1. 点击"新建会话"
2. 输入会话名称
3. 选择要使用的角色（可以多选）
4. 点击"创建"
5. 聊天页面会自动打开
```

---

### 2. 角色标签（重要！）

**功能**：
- 创建AI角色
- 配置AI平台
- 测试连接

**操作**：
```
1. 点击"角色"标签
2. 点击"新建角色"
3. 填写信息：
   - 名称：DeepSeek助手
   - 服务提供商：DeepSeek ← 这里选择AI平台
   - 模型：deepseek-chat
   - 系统提示词：你是一个友好的助手
4. 点击"创建"
5. 点击"测试"按钮验证连接
```

**支持的AI平台**：
- DeepSeek
- 豆包
- 千问
- ChatGPT

---

### 3. 设置标签

**功能**：
- 查看使用说明
- 了解支持的AI平台
- 重要提示
