## 🚀 现在你可以这样操作

### 方法1：最简单（推荐！）

```
1. 打开侧边栏
2. 点击"角色"标签
3. 点击"新建角色"
4. 填写：
   角色名称：DeepSeek助手
   服务提供商：DeepSeek ← 选择这个
   模型：（留空！会自动填充） ← 不用填！
   系统提示词：（可选，也可以留空）
5. 点击"创建"
```

**就这么简单！** 模型字段会自动填充为 `deepseek-chat`

---

### 方法2：手动指定模型

如果你想使用特定模型：

```
1. 选择服务提供商：DeepSeek
2. 模型会自动填充为：deepseek-chat
3. 如果想用其他模型，可以手动修改：
   - deepseek-chat（通用对话）
   - deepseek-coder（编程专用）
```

## 🎯 完整操作流程

```
1. 打开 edge://extensions/

3. 点击刷新按钮 🔄

```


```
1. 打开侧边栏（Ctrl+Shift+S）

3. 点击"新建角色"
4. 填写：
   角色名称：DeepSeek助手
   服务提供商：DeepSeek
   模型：（留空！）
   系统提示词：你是一个友好的助手
5. 点击"创建"
```

### 第4步：测试连接

```
1. 在角色列表找到"DeepSeek助手"
2. 点击"测试"按钮
3. 等待结果
```

```
1. 点击"会话"标签
2. 点击"新建会话"
3. 勾选"DeepSeek助手"
4. 点击"创建"
5. 开始对话
```
